2026 MCM Problem C: Data With The Stars (DWTS) 详细任务书

1. 问题背景与核心冲突
   美国电视节目《与星共舞》（Dancing with the Stars, DWTS）是一场名人与专业舞者搭档的舞蹈比赛。比赛结果由两部分决定：

专业评委（Judges）： 评分基于技术动作，分值 1-10 分 。

观众粉丝（Fans）： 投票基于喜爱程度、个人魅力或舞蹈质量，票数保密 。

核心问题： 如何将“评委分”和“观众票”结合来决定淘汰谁？历史上节目组曾多次修改规则，因为出现了“技术很差但人气极高”的选手（如 Jerry Rice, Bobby Bones）走得太远甚至夺冠的争议情况 。

2. 必须掌握的计分机制 (Game Mechanics)
   你的模型必须理解并能复现以下三种历史机制：

机制 A：排名加和法 (Combined by Rank)

适用时间： 第1-2季，以及第28-34季（推测） 。

算法：

将选手的评委分排名（第1名，第2名...）。

将选手的观众票数排名。

总分 = 评委排名 + 观众排名。

总分最低者为本周第一，总分最高者被淘汰 。

特点： 排名是线性的，分差大小不重要，只看相对位置。

机制 B：百分比加和法 (Combined by Percent)

适用时间： 第3-27季 。

算法：

评委得分率： (该选手评委分 / 所有选手评委总分) × 100%。

观众得票率： (该选手观众票数 / 所有选手观众总票数) × 100%。

总分 = 评委得分率 + 观众得票率。

总分最高者晋级，最低者淘汰 。

特点： 对分差敏感。如果评委分极低，需要巨量的观众票比例才能拉回分数。

机制 C：评委拯救环节 (Judges' Save)

适用时间： 第28季及以后（作为机制A的补充） 。

算法： 先通过“排名加和法”算出倒数两名（Bottom Two）。然后，评委在直播中直接投票决定这两组中谁留下，谁淘汰。这意味着倒数第二名可能会被评委“救”回来，而淘汰倒数第一名 。

3. 建模任务清单 (Detailed Modeling Tasks)
   你需要利用提供的 2026_MCM_Problem_C_Data.csv 数据集完成以下四个步骤：

任务一：重建“幽灵”数据 (Reconstruct the Unknown)
由于观众投票数是未知的（Secret），你需要建立一个数学模型来反向推导每一季、每一周各选手的观众投票分布。

输入： 评委分数、每周被淘汰的选手名单、当时的计分规则（Rank 或 Percent）。

目标： 生成一套“估算的观众投票数”。

验证： 你的估算数据必须能解释为什么当时那个人被淘汰了 。

量化不确定性： 你的估算有多准？是否存在多种投票组合都能导致同一结果？请给出“确定性度量”（Certainty Measure） 。

任务二：平行宇宙实验 (Counterfactual Analysis)
利用你反推出来的观众投票数据，进行“如果当时……会怎样”的分析：

全赛季对比： 如果第1-2季用百分比法，第3-27季用排名法，结果会有什么不同？哪种方法更偏向观众意愿？ 。

争议个案分析： 针对以下具体案例，分析更换计分方法或加入评委拯救环节是否会改变他们的命运？

Jerry Rice (S2): 评委分极低却获亚军。

Billy Ray Cyrus (S4): 评委分垫底却获第5名。

Bristol Palin (S11): 12次评委分最低却获第3名。

Bobby Bones (S27): 评委分低却夺冠。

推荐： 基于分析，未来赛季应该用哪种方法？是否应该保留“评委拯救环节”？ 。

任务三：成功因素分析 (Factor Analysis)
结合你估算的投票数据和已知数据，建立模型分析以下因素的影响力 ：

专业舞伴 (Pro Dancers): 某些舞伴是否更容易带飞明星？

明星特征 (Celebrity Attributes): 年龄、行业（运动员vs演员）、家乡/国家。

差异性： 这些因素对“评委打分”和“观众投票”的影响是否不同？（例如：长得好看是否只影响观众票，不影响评委分？）

任务四：设计新赛制 (System Design)

目标： 提出一套全新的、或者改进的计分系统 。

标准： 你的系统必须比现有的更“公平”，或者更能平衡娱乐性与专业性。

论证： 你必须向节目制作人证明，为什么要采纳你的新方案 。

4. 数据处理指南 (Data Guidelines)
   在使用 2026_MCM_Problem_C_Data.csv 时，请注意以下细节：

分数结构： 评委分为 1-10。若出现小数（如 8.5），是因为多支舞蹈取了平均分 。

特殊数值：

0：代表该选手已被淘汰，不再参赛 。

N/A：代表该周没有第4位评委，或该赛季周数较短 。

全明星赛季： 第15季是全明星赛（All-Star），选手都是以前参赛过的，分析时需注意样本的特殊性 。

外部数据： 你可以自行补充外部数据（如选手的社交媒体粉丝数、当时的新闻热度等），但必须注明来源 。
